Project intended for Code Challenge for sample service. 

Project based on Maven + Jersey + jquery + json style.

Run the below command to set the project. 

Step 1.Run Maven clean install -DskipTests since test case looks for deployed service at project level.

Step 2. Run cobertura:cobertura to get code coverage. 

Step 3: Use IDE (say eclipse) to pull the project and start your development

Step 4 : For Maven Eclipse web project use maven command eclipse:eclipse -Dwtpversion=2.0
